# Publicación en GitHub Pages

1. Descomprima este paquete.
2. Copie todo el contenido en el repositorio `juancreyeshn-design.github.io` o en la carpeta configurada para GitHub Pages.
3. Verifique que `index.html` quede en la raíz publicada.
4. Haga commit y push.
5. Abra la URL pública del repositorio.

La carpeta `offline/` contiene el HTML único portable.
